# quiz-builder

for fill-in-the-blank style quizzes, where the blanks are dropdown inputs.

Written with React.js and Flask with a sqlite database.

### install & run

1. in `web`:
    * `npm ci`
    * `npm start`
2. in `api`:
    * `pip install -r requirements.txt` (or `pip install Flask`)
    * `python db/init.py`
    * `flask run`
